# Corne-ish Zen V2 Layout Reference

## Base Layer (COLEMAK)
```
┌─────┬─────┬─────┬─────┬─────┬─────┐       ┌─────┬─────┬─────┬─────┬─────┬─────┐
│  `  │  Q  │  W  │  F  │  P  │  G  │       │  J  │  L  │  U  │  Y  │  ;  │  -  │
├─────┼─────┼─────┼─────┼─────┼─────┤       ├─────┼─────┼─────┼─────┼─────┼─────┤
│Ctrl │A/Ctl│R/Alt│S/GUI│T/Sft│  D  │       │  H  │N/Sft│E/GUI│I/Alt│O/Ctl│  '  │
├─────┼─────┼─────┼─────┼─────┼─────┤       ├─────┼─────┼─────┼─────┼─────┼─────┤
│  (  │  Z  │  X  │  C  │  V  │  B  │       │  K  │  M  │  ,  │  .  │  /  │  )  │
└─────┴─────┴─────┼─────┼─────┼─────┤       ├─────┼─────┼─────┼─────┴─────┴─────┘
                  │Vol  │Num  │Mouse│       │Enter│Sym  │Uni  │
                  │Esc  │Space│ Tab │       │     │Bksp │ Del │
                  └─────┴─────┴─────┘       └─────┴─────┴─────┘
```

**Home Row Modifiers:**
- A/Ctrl, R/Alt, S/GUI, T/Shift (left hand)
- N/Shift, E/GUI, I/Alt, O/Ctrl (right hand)

**Layer Access:**
- Volume (3): Hold Esc
- Number (1): Hold Space  
- Mouse (4): Hold Tab
- Symbol (2): Hold Backspace
- Unicode (7): Hold Del

```
┌─────┬─────┬─────┬─────┬─────┬─────┐       ┌─────┬─────┬─────┬─────┬─────┬─────┐
```

```
├─────┼─────┼─────┼─────┼─────┼─────┤       ├─────┼─────┼─────┼─────┼─────┼─────┤
│Shift│ž/Ž │     │č/Č │     │     │       │     │     │     │     │     │Shift│

```

**Czech Characters:**
- Tap: lowercase (á, ř, š, etc.)
- Hold: uppercase (Á, Ř, Š, etc.)
- Special: é (tap) / ě (hold), ú (tap) / ů (hold)
- Shift keys in corners for additional capitals

## Combos (Key Combinations)

### Copy/Paste Combos
```
Bottom Row: Z  X  C  V  B
           ┌─┬─┐
           │Z│X│ = Ctrl+C (Copy)
           └─┴─┘
              ┌─┬─┐
              │C│V│ = Ctrl+V (Paste)
              └─┴─┘
           ┌─┬─┬─┐
           │Z│X│C│ = Ctrl+Shift+C (Copy with formatting)
           └─┴─┴─┘
              ┌─┬─┬─┐
              │X│C│V│ = Ctrl+Shift+V (Paste without formatting)
              └─┴─┴─┘
```

## Quick Reference

**Layer Access:**
- Hold Esc → Volume/Media
- Hold Space → Numbers/Navigation  
- Hold Tab → Mouse
- Hold Backspace → Symbols/F-keys
- Hold Del → Czech Unicode

**Special Features:**
- Caps Word: Number layer + left pinkie
- Gaming: Number layer + right pinkie
- Bluetooth: Number layer + bottom row

**Czech Typing:**
1. Hold Del for Unicode layer
2. Tap for lowercase, hold for uppercase
3. Use corner Shift keys for additional capitals
